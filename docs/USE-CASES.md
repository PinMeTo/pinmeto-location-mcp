# Use Cases & Example Prompts

This guide provides example prompts organized by business need. Simply copy these prompts or use them as inspiration for your own questions.

---

## Performance Monitoring

Track how your locations are performing across platforms.

### Ratings Overview
```
What's my average Google rating across all locations?
```

```
Show me locations with Google ratings below 4 stars
```

```
Which locations have the highest ratings on Facebook?
```

### Traffic & Engagement
```
How many people viewed my locations on Google last month?
```

```
Show me direction requests for all my Swedish locations
```

```
Compare website clicks this quarter to last quarter
```

### Platform Comparison
```
How does my Apple Maps performance compare to Google?
```

```
Which platform drives the most direction requests?
```

## Tips for Better Prompts

### Be Specific About Time
- Use specific dates: "from January 1 to March 31, 2024"
- Use relative periods: "last month", "this quarter", "year-to-date"

### Specify Locations When Needed
- Single location: "for store 1234"
- By city: "for my Stockholm locations"
- By country: "across all Swedish locations"

### Ask for Comparisons
- "Compare to last month"
- "Year-over-year change"
- "Versus the same period last year"

### Request Specific Formats
- "Show me as a table"
- "Give me the top 10"
- "Summarize the key findings"
